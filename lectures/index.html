What is a Variable?

A variable is a bucket

with a name.

Let's work through an example. 

						int buck;
					

						int buck;
int indicates that the variable will hold an 'integer' (i.e. a whole number, no decimals). 

						int buck;
buck is the name of the variable. 

						int buck;

In C# commands are terminated with semicolons. Think of it like the period at the end of a sentence.

Let's say we want to do something with our variable. How about we try to print it to the console?


						int buck;
						Debug.Log(buck);
A variable is a bucket or container. Here it has the name buck.


						int buck;
						Debug.Log(buck);

When we use its name elsewhere we are saying:

take whatever is inside buck and use it here.

In this case we printed it to the console.


						int buck;
						Debug.Log(buck);
						

						0
Where did the 0 come from?

Whenever possible, a newly declared variable will be assigned a default value. What that default value is depends on the type of the variable.

In the case of integers, (and most other types of numbers) the default value is `0`. So when we print it to the console, we get `0`.

If we want to store some other value in a variable, we need to assign it


						buck = 6;

put the number 6 into buck

the old value (0) is overwritten.

What else can we do with variables?
Addition: 

						int a = 0;
						int b = a + 1;

						Debug.Log("a: " + a);
						Debug.Log("b: " + b);
					

						a: 0
						b: 1
Subtraction: 

						b = a - 2;

						Debug.Log("b: " + b);
					

						b: -2
Multiplication: 

						int c = b * 3;

						Debug.Log("c: " + c);
					

						c: -6
Multiplication: 

						int d = c * b;

						Debug.Log("d: " + d);
					

						d: 12
Division: 

						int e = d / 4;

						Debug.Log("e: " + e);
					

						e: 3
